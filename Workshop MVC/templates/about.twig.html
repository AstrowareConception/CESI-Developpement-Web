À propos de My Todoo

My Todoo est une application web de gestion de tâches (To-Do List) développée dans un cadre pédagogique pour illustrer les concepts fondamentaux du développement web moderne en PHP.

🚀 Objectifs du projet

  • Mise en œuvre d'une architecture MVC (Modèle-Vue-Contrôleur).
  • Gestion des dépendances avec Composer.
  • Utilisation du moteur de template Twig pour une séparation nette entre logique et présentation.
  • Développement d'une interface responsive et accessible.

🏗️ Architecture

L'application est structurée de la manière suivante :

  • index.php : Le Front Controller (routeur) qui reçoit toutes les requêtes.
  • src/Models : Gère l'accès aux données (ici, une base de données fichier CSV).
  • src/Controllers : Contient la logique de traitement et fait le pont entre le Modèle et la Vue.
  • templates/ : Contient les fichiers Twig pour le rendu HTML.
  • static/ : Regroupe les ressources statiques (CSS, images).

💡 Concepts clés démontrés

Autoloading : Grâce au standard PSR-4 configuré dans Composer, les classes sont chargées automatiquement sans avoir besoin de require manuels à chaque fichier.

Héritage de templates : Le fichier layout.twig.html sert de structure commune à toutes les pages, évitant la duplication de code HTML (DRY - Don't Repeat Yourself).

Variables globales : L'URL de base est injectée globalement dans Twig pour faciliter la gestion des liens.

Réalisé avec ❤️ par l'équipe pédagogique du CESI.
